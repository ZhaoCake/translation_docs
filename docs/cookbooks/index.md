# 翻译文献 - Chisel Cookbooks

Chisel硬件设计语言的Cookbooks中文文档。

## 什么是Chisel？

Chisel（Constructing Hardware In a Scala Embedded Language）是一种开源的硬件设计语言，它嵌入在Scala编程语言中，用于描述数字电路。Chisel提供了抽象的、高级的硬件构造，使硬件设计更加模块化、可重用和可测试。
